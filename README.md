# hypr_dots


Distro - Arch linux  
WM - Hyprland  
notify - mako  
bar - waybar  
terminal - foot   
### Installation
```bash
sudo pacman -S ttf-jetbrains-mono ttf-jetbrains-mono-nerd mako foot waybar hyprland
```

```bash
git clone https://github.com/hyuka1/hypr_dots.git
cd hypr_dots
cp -r .config/* ~/.config/
```
